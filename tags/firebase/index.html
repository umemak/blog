Gitpodでfirebaseログインする方法

2020-04-08

何度もググるので。。

$ npm i -g firebase-tools
$ firebase login --no-localhost

表示されるURLを別タブで開くと認証画面が表示されるので許可する。 許可するとコードが表示されるのでコピーしてコマンドラインに貼り付ける。

firebase gitpod
