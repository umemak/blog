Flutter 3

2022-05-12

［速報］「Flutter 3」登場、FlutterによるmacOSとLinuxアプリ開発が正式版に。Macのユニバーサルバイナリにも対応 － Publickey

MacとLinuxのアプリケーション開発の正式版より、「Firebase plugin for Flutter」のほうが気になった。

これは前に使っていたFlutterFireとは別物？と思ったけど、FlutterFireの説明が「The official Firebase plugins for Flutter.」ってなってるし、https://firebase.flutter.dev/がhttps://firebase.google.com/docs/flutter/setupにリダイレクトされるようになっているので、FlutterFireがFirebase plugin for Flutterに改名したということなのかな。
