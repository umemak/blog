gitが入ってなかったのでインストール。

$ sudo yum update
$ sudo yum install git
$ git --version
git version 2.32.0

npmも入っていなかった。 チュートリアル: Amazon EC2 インスタンスでの Node.js のセットアップ - AWS SDK for JavaScript

$ . ~/.nvm/nvm.sh
