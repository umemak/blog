昨日洗い出した画面をとりあえず書いてみる。

画面遷移にvue-routerを使うので、インストールしておく。
```
npm install vue-router
```
各画面は、`src/views/`に置く。

```
 $ ls -1 src/views/

EntryConfirm.vue
EventCreate.vue
EventDetail.vue
EventEdit.vue
EventSearch.vue
Login.vue
MyPage.vue
Top.vue
```

ファイル作るところまでで今日は時間切れ。。
