WSLのmakeでmkdir

2022-10-14

.PHONY: mkdir
mkdir:
	mkdir -p ./work/{a,b,c}

$ make mkdir 
mkdir -p ./work/{a,b,c}

$ ls work
{a,b,c}

なんでや。

$ rm -r work

$ mkdir -p ./work/{a,b,c}
$ ls work
a  b  c

こうなってほしい。

→make内では、デフォルトのシェルが/bin/shになっているので、MakefileでSHELL=/bin/bashを書く必要がある。

  • とあるエンジニアの備忘log: Make のポータビリティについて考える
  • Makefileには使用するshellを定義すると良さそう - Qiita
